# My Portfolio - S M Ehasanul Haque

This is the repository for my personal portfolio website, showcasing my skills, projects, and experience as a Full-Stack Web Developer.

**Live Demo:** [ehasanlive.com](https://ehasanlive.com)

## Introduction

A passionate software engineer specialized in building production-grade applications and fluid interactive interfaces. I architect secure backends with Python/Django and craft pixel-perfect frontends using React and Tailwind. My focus is on delivering high-performance, scalable solutions that solve real-world problems.

## Features

*   **Dynamic Sections:** The portfolio is divided into multiple sections: About, Services, Skills, Education, Experience, Projects, and Contact.
*   **Bilingual Support:** The content can be switched between English and Bengali.
*   **Interactive Chatbot:** An AI-powered chatbot to answer visitor queries in real-time.
*   **Theme Customization:** A theme settings panel to change the website's color scheme.
*   **Smooth Animations:** Uses Framer Motion for smooth and fluid animations.
*   **Responsive Design:** The website is fully responsive and works on all screen sizes.

## Technologies Used

*   **Frontend:**
    *   React
    *   Vite
    *   Tailwind CSS
    *   Framer Motion
*   **Backend (for the chatbot):**
    *   Node.js (implicitly, for the OpenAI API interaction)
*   **APIs:**
    *   EmailJS (for the contact form)
    *   OpenAI API (for the chatbot)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js
*   npm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Ehasan07/my-protfolio.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```
